# yt-dlp Binary

Download the yt-dlp binary for your platform from:
https://github.com/yt-dlp/yt-dlp/releases/latest

Place it here with the correct name for your platform:

- Windows: `yt-dlp-x86_64-pc-windows-msvc.exe`
- Linux:   `yt-dlp-x86_64-unknown-linux-gnu`
- macOS ARM: `yt-dlp-aarch64-apple-darwin`
- macOS Intel: `yt-dlp-x86_64-apple-darwin`

The app will also fall back to checking your system PATH for yt-dlp,
so if you already have it installed globally, you don't need to put it here.
